Project Details

The realism match mutator is the core mutator of all Dog Company drills. It is used to handle all of the match mode and scrimmage admin functionality. In Fall 2021 I started maintaining and updating it with new features.

General Commands

  • mutate realismhelp - Displays the help menu

  • mutate respawn - Respawns the player

  • mutate changename - Changes the player's name

  • mutate safetyoff - Disables the player's firing button

  • mutate safetyon - Enables the player's firing button

Scrim Admin Commands

  • mutate scrimadminhelp - Displays a help menu of admin commands

  • mutate swapteams - Forces a team swap

  • mutate swapnorth - Swaps the north team to the south

  • mutate swapsouth - Swaps the south team to the north

  • mutate weaponshold - Turns on the weapons safeties of all players

  • mutate weaponsfree - Turns off the weapon safeties of all players

  • mutate respawnall - Respawns all dead players

  • mutate forcechangename [CURRENTNAME to NEWNAME] - Changes the name of the specified player

  • mutate whois [NAME] - Displays the steam64 ID and original name of the specified player

Match Control Commands

  • mutate enablematch - Enables match mode

  • mutate disablematch - Disables match mode

  • mutate matchlive - Forces the match LIVE

  • mutate resetlive - Cancels LIVE

  • mutate restartround - Restarts the round

  • mutate endround - Suicides all players to end the round

Objective Commands

  • mutate openobj [OBJLETTER] - Opens the specified objective, specify ALL to open all objectives

  • mutate closeobj [OBJLETTER] - Closes the specified objective, specify ALL to close all objectives

  • mutate setobjneutral [OBJLETTER] - Sets the specified objective neutral, specify ALL to set all objectives

  • mutate setobjnorth [OBJLETTER] - Sets the specified objective north owned, specify ALL to set all objectives

  • mutate setobjsouth [OBJLETTER] - Sets the specified objective south owned, specify ALL to set all objectives

Match Setup Commands

  • mutate setroundduration [MINUTES] - Sets the round timer to the specified time

  • mutate setff [SCALE 0 - 1] - Sets the friendly fire modifier to the specified decimal

  • mutate setnorthreinforcements [AMOUNT] - Sets the north ticket count to the specified amount

  • mutate setsouthreinforcements [AMOUNT] - Sets the south ticket count to the specified amount

  • mutate toggleautorespawns - Toggles auto respawns during briefing

Chat Commands

  • /ready - Sets the players team ready

  • /notready - Sets the players team not ready

  • /AIT - Displays how many AIT slots are being used by both teams

  • /alive - Displays how many are players alive on each team
